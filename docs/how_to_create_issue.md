# 문제 출제 방법

### 1. 저장소 상단 `Issues` 클릭

### 2. `New issue` 버튼 클릭

### 3. 미리 작성된 template을 바탕으로 내용 작성

- Title : [날짜] 문제플랫폼 : 문제
  - ex) [2022.02.10(목)] BOJ(2839) : 설탕 배달
- 문제 링크 : 해당 문제의 링크를 작성
- 난이도 : 해당 문제의 난이도가 존재한다면 같이 적어주세요(마음의 준비를 위해...)
  - ex) solved.ac 티어, 프로그래머스 level

### 4. `Submit new issue` 버튼 클릭
